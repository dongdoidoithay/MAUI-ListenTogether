### 仓库介绍  
该仓库整合了一些常用的音乐平台接口。
* 支持 网易、酷狗、酷我、咪咕 数据源。
* 不支持登录对应平台账号。
* 程序只是整合了不同平台的音乐链接，当一个平台听不了时你可以方便的去另一个平台再碰碰运气🐶🐶  

### 使用  
* 所有功能的入口都在 `MusicNetPlatform` 类，方法命名应该还算通俗易懂，客官自己研究一下吧。  
* **使用前先进行组件初始化 `MusicNetPlatform.InitializeAsync()`**
